ÖZELLIKLER

Bir plugin'den fazlası.

Salt AI Translator, basit bir çeviri aracı değil. İçerik yönetimi, SEO, toplu işlem ve export/import ile tam anlamıyla bir çok dilli site altyapısıdır.

🔄

Round-Robin API Key

Birden fazla API key tanımlayın — plugin sırayla kullanarak rate limit yükünü dağıtır. 429 gelince Retry-After header'ına göre bekler, exponential backoff ile yeniden dener.

RATE LIMIT SAFE
⚡

Background Queue

Action Scheduler veya WP Cron ile arka planda çalışır. Yüzlerce post'u kuyruğa ekleyin, siteniz yavaşlamadan çeviriler tamamlansın. Driver badge ile hangisinin çalıştığını görün.

ASYNC
📝

Gutenberg Block Çevirisi

Gutenberg blokları ayrıştırılarak yalnızca innerContent çevrilir, block yapısı ve HTML korunur. İki kat daha az API çağrısı ile %50 maliyet tasarrufu.

COST OPTIMIZED
🔍

SEO Meta Çevirisi

Yoast SEO ve Rank Math ile entegre. Meta description, SEO title, Open Graph tag'leri ve görsel alt text'leri otomatik çevrilir veya AI ile oluşturulur.

YOAST + RANKMATH

Post, term, menü, ACF alanları ve Polylang stringlerini tek XLSX'e export edin. Import ile batch güncelleme yapın. Dry-run modu ile önce farkı görün, sonra kaydedin.

BATCH IMPORT

NASIL KULLANILIR

5 adımda hazır.

1

Plugin'i aktive edin

WordPress admin'de Plugins → Add New → "Salt AI Translator" yükleyin ve aktive edin.

2

Translator seçin

Settings → Translator sekmesinde OpenAI, DeepL, Claude veya diğerlerinden birini seçin. Birden fazla API key ekleyebilirsiniz — round-robin otomatik çalışır.

3

Çok dilli plugin'i bağlayın

Polylang, WPML veya qTranslate-XT kuruluysa Salt AI otomatik algılar ve entegre olur. Ek yapılandırma gerekmez.

4

Tek post çevirin

Post editöründe sağ sidebar'daki Salt Translate kutusundan dil seçip Translate'e basın. Saniyeler içinde hazır.

5

Toplu çeviri yapın

Posts sayfasından dil ve post type seçin → Check Untranslated → Translate All. Yüzlerce post arka planda sıraya alınır.

sat_settings — wp_options
// Settings yapısı (sat_settings)
[
  'translator' => 'openai',
  'model' => 'gpt-4o-mini',
  'temperature' => '0.2',
  'api_keys' => [
    'openai' => [
      'sk-proj-...',
      'sk-proj-...', // round-robin
    ],
  ],
  'translate_slugs' => 1,
  'auto_translate' => 0,
  'glossary' => [
    [
      'source' => 'WordPress',
      'target' => '',
    ],
  ],
  'seo' => [
    'meta_desc' => [
      'translate' => 1,
    ],
  ],
]

ENTEGRASYONLAR

Mevcut stack'inize uyum sağlar.

🟢

Polylang & Polylang Pro

Post, term, menü ve string çevirisi. PLL_MO ile Polylang string API'ye doğrudan yazma. String group, skip translated, batch queue tam destek.

🔵

WPML

Translatable post type / taxonomy detection, setContext, translate_slugs ve untranslated post ID filtresi. Canlı test aşamasında.

🟣

qTranslate-XT

buildQtranslateField ile çok dilli alan yapısı oluşturma. Closing tag çakışması fix'i dahil.

🟡

Yoast SEO & Rank Math

Meta description, SEO title, OG tags ve focus keyword çevirisi. Her iki eklenti için ayrı interface implementasyonu.

🔴

Advanced Custom Fields

Text, textarea ve wysiwyg ACF alanlarını post çevirisiyle eş zamanlı çevirin. AcfTranslator sınıfı ile native entegrasyon.

🩵

Action Scheduler

WooCommerce veya standalone Action Scheduler varsa otomatik kullanılır. Her queue item izlenebilir, başarısız item'lar otomatik retry.
